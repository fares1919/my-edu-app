# Fonts — Noto Sans Arabic / Cairo

These fonts must be downloaded and placed in this directory for offline use.

## Download

### Noto Sans Arabic (primary)
https://github.com/notofonts/noto-fonts/tree/main/hinted/ttf/NotoSansArabic

Required weights:
- NotoSansArabic-Regular.ttf
- NotoSansArabic-SemiBold.ttf  
- NotoSansArabic-Bold.ttf

### Cairo (optional / dyslexic mode)
https://github.com/Gue3bara/Cairo

Required weights:
- Cairo-Regular.ttf
- Cairo-SemiBold.ttf
- Cairo-Bold.ttf

Place all font files in this directory (`public/fonts/`).

The app will fall back to Google Fonts CDN if local files are not found.
